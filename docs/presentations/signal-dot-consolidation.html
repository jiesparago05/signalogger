Signalog / Signal Dot Consolidation

Feature Design

Signal Dot Consolidation

Auto-merge nearby signal readings on the map to reduce clutter from daily commuters traveling the same routes.

📍

~50m Grid

Readings within the same ~50 meter grid cell get grouped together. Coordinates rounded to 0.0005 degrees.
⏱

24h Threshold

Only readings older than 24 hours are consolidated. Fresh readings always show as individual dots.
📡

Same Carrier + Type

Only merges readings with the same carrier AND network type (e.g., Smart 4G stays separate from Globe 4G).
📶

Cellular Only

WiFi readings are excluded from consolidation. Only cellular signal data gets merged.

Grid Snapping Formula

Summary

Key Decisions

Grid-based, not radius-based
Rounding to 0.0005 degrees is fast, indexable, and doesn't require distance calculations.
Originals kept, not deleted
Original readings are marked as consolidated but preserved for drill-down history.
Local-first consolidation too
Same algorithm runs client-side on AsyncStorage data, so the map stays clean even offline.
Per-carrier separation
Smart 4G and Globe 4G at the same spot remain as separate consolidated dots for accurate comparison.
